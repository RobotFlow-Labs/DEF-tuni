# PRD-001 — Environment Setup

## Objective
Create a reproducible local environment with Python 3.11, `uv`, and CUDA dependency stack for TUNI integration.

## Steps
1. Run `bash scripts/bootstrap_uv.sh`.
2. Validate editable install with `def-tuni --help`.
3. Confirm upstream repo exists at `repositories/TUNI`.

## Done Criteria
1. `.venv` exists with Python 3.11.
2. `def-tuni eval --dataset FMB --dry-run` runs.
3. `def-tuni fps --dry-run` runs.
