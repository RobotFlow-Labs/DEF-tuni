# PRD-003 — Baseline Evaluation Wrappers

## Objective
Run TUNI paper evaluation scripts through a single module CLI for FMB, PST900, and CART.

## Steps
1. CUDA mode:
   - `ANIMA_BACKEND=cuda def-tuni eval --dataset FMB --checkpoint /path/to/model.pth`
   - `ANIMA_BACKEND=cuda def-tuni eval --dataset PST900 --checkpoint /path/to/model.pth`
   - `ANIMA_BACKEND=cuda def-tuni eval --dataset CART --checkpoint /path/to/model.pth`
2. MLX mode scaffold:
   - `ANIMA_BACKEND=mlx def-tuni eval --dataset FMB`
3. Capture outputs and failures in `benchmarks/`.

## Done Criteria
1. CLI dispatches to correct dataset script and weight argument.
2. Missing checkpoint errors are explicit for CUDA mode.
3. MLX mode executes stable scaffold path without crashing.
